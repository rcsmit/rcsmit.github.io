Auth & Security

Platform-wide
Single identity, event-scoped roles
One login. Memberships at multiple events across tenants. Roles resolved per-request from DB — not in JWT.
JWT tokens
15-min access tokens, 7-day refresh tokens with rotation and family revocation.
Auth
Magic links
Single-use, 15-min expiry, timing-safe comparison, rate-limited.
TOTP 2FA
Mandatory for Organizers and SiteOwners. Optional for others. Backup codes. Full reset chain of authority.
Rate limiting
Per endpoint, Redis-backed, sliding window.
Webhook signature verification
Stripe and Mollie — verified on every inbound payment event.
File upload security
MIME validation, EXIF stripping, malware scan, no original filenames preserved.
Tenant isolation
Prisma middleware + service-layer explicit filters. Violations logged as security events.
Invariant
Soft delete
404 to all non-admin callers. Explicit ?includeDeleted=true for organizers.

Platform Infrastructure

SiteOwner / DevOps
Multi-tenant PostgreSQL
Shared DB, all records tenantId-scoped. Upgrade path to per-tenant schemas without redesign.
Plan limit enforcement
Enforced at 6 layers: API, UI, BullMQ worker, storage upload handler, website publish trigger, data export endpoint.
Invariant
Async job queue
BullMQ + Redis. Retry policies, dead-letter queues, deduplication, poison message handling.
Observability
Pino structured logs, OpenTelemetry traces, Sentry error capture, health + readiness endpoints.
Circuit breaker
Per payment provider. Redis-shared state. 503 on open.
Redis caching
Intentionally coarse tenant-level invalidation. SCAN-based batch delete. Pre-warming for homepage and programme.
Backups
Nightly pg_dump to S3. 30-day retention. Documented restore procedure.
Timestamps
UTC in DB. Displayed in event's IANA timezone. Always.
