About AcroInsiders

What is this?

An interactive world overview of Acroyoga events, retreats, festivals and intensives.

Use the filter bar to narrow events by date, continent, country or event type. Click any event to see full details.

How to use

  • Click any event card to see full details
  • Filter by continent and country using the dropdowns
  • Toggle event types using the pill buttons
  • Use date pickers to narrow to a specific period
  • Past (ended) events are hidden by default
  • Switch to Calendar for a monthly overview
  • Switch to Map to see events on a world map

About the map

Map pins are geocoded from city and country name — they show the approximate area, not the exact venue address.

  • Small towns or villages may be pinned to the nearest known city
  • Events with missing or unrecognisable location data do not appear on the map at all
  • Always check the event details page for the precise venue
